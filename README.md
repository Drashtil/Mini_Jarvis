# Mini Jarvis

A Python-based virtual assistant inspired by Jarvis that can perform basic tasks using voice or text commands.

## Features

* Respond to user commands
* Open websites and applications
* Search information online
* Tell time and date
* Simple voice assistant functionality

## Technologies Used

* Python
* SpeechRecognition
* pyttsx3
* Webbrowser Module

## How It Works

The program:

1. Takes voice or text commands from the user
2. Processes the command
3. Performs the requested action
4. Responds with voice or text output
